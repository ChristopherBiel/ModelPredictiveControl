## How to:
To start:
1. Setup a virtual environment in python, e.g. by using:  
`python3.8 -m venv venv` on  Ubuntu or MacOS
`py -3.8 -m venv venv` on Windows
2. Execute the install_deps.py file of the assignment
3. Enjoy!
